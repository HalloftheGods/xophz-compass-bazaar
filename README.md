# Xophz Bazaar Foresight

> **Category:** Command Deck · **Version:** 1.0.0

A supercharged UI for managing WooCommerce inventory, orders, and viewing sales data in real time.

## Description

**Bazaar Foresight** is the commerce command center for COMPASS. It wraps WooCommerce's sprawling admin into a streamlined, glassmorphic dashboard where store operators can manage products, monitor orders, and track revenue — all from a single, responsive interface.

### Core Capabilities

- **Inventory Management** – View and edit products, stock levels, and pricing in a modern data grid.
- **Order Dashboard** – Real-time order feed with status filtering, search, and quick actions.
- **Sales Analytics** – Revenue charts, conversion metrics, and trend data at a glance.

## Requirements

- **Xophz COMPASS** parent plugin (active)
- **WooCommerce** (active)
- WordPress 5.8+, PHP 7.4+

## Installation

1. Ensure **Xophz COMPASS** and **WooCommerce** are installed and active.
2. Upload `xophz-compass-bazaar` to `/wp-content/plugins/`.
3. Activate through the Plugins menu.
4. Access via the My Compass dashboard → **Bazaar**.

## Frontend Routes

| Route | View | Description |
|---|---|---|
| `/bazaar` | Dashboard | Inventory overview, order feed, and sales charts |
